# Beaucharme Cosmétique

Site e-commerce pour Beaucharme Cosmétique, une marque de cosmétiques naturels créée par deux agriculteurs bourguignons près de Vézelay.

## Caractéristiques

- **Design moderne et authentique** inspiré par Aesop, Le Labo et Typology
- **Stack technique** : Astro + Tailwind CSS
- **E-commerce mock UI** avec panier fonctionnel (localStorage)
- **4 pages principales** : Accueil, Notre Histoire, Boutique, Contact
- **Responsive design** : Desktop-first avec adaptation mobile soignée
- **30+ produits** organisés en 6 catégories

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:4321`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Fonctionnalités

### Panier
- Ajout/suppression de produits
- Modification des quantités
- Persistance avec localStorage
- Interface sidebar élégante

### Boutique
- Filtres par catégorie
- Grille de produits responsive
- Badges pour produits en rupture
- Hover effects subtils

### Design
- Palette de couleurs naturelles (beige, terracotta, sage, gold)
- Typographie élégante (Cormorant Garamond + Inter)
- Animations CSS fluides
- Grain texture subtile

## Brand Identity

**Promesse** : De la graine au flacon — traçabilité totale

**Points différenciants** :
- Intégration verticale totale
- Plantes rares cultivées en France
- 75% minimum d'ingrédients de la ferme
- Certification COSMOS
- Packaging éco-conçu
